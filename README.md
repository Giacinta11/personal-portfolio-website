# Personal Portfolio Website

## Overview
This project is a personal portfolio website designed to present my professional profile, skills, and projects.

The website provides a clear and responsive user experience across desktop, tablet, and mobile devices.

## Features
- Responsive design (mobile-friendly)
- Home page with personal introduction
- Portfolio section showcasing projects
- Individual project pages
- Contact page with contact form and social links
- Navigation menu for easy browsing

## Technologies
- HTML
- CSS
- JavaScript
- Bootstrap

## Structure
- **Home**: introduction and overview
- **Portfolio**: list of projects
- **Project Page**: detailed view of each project
- **Contact**: contact form and personal links
